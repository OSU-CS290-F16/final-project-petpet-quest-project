PetPet Quest is a click based game similar in mechanic to games like Cookie Clicker. On sign up user gets one free pet, currently a cat. By clicking on a pet in the sidebar the home background is replaced with that pet. The pet displayed can be changed by choosing a different pet from the side bar. By moving the mouse over the image of the pet, thus petting it, the user gains points. These points can then be used to buy more pets from the store page. If the user already owns a pet the pet will be shown as SOLD in the store. The goal is to own every pet.
